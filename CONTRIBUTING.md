# Contributing

Thanks for helping build AITestAutomate.

## Development Setup

```bash
npm install
npm run dev:api
npm run dev:web
```

Open `http://localhost:5173`.

## Useful Commands

```bash
npm run check
npm run build
```

## Contribution Areas

Good first areas:

- Add HAR parsing examples.
- Improve endpoint grouping.
- Add API assertion types.
- Improve failure explanations.
- Add documentation screenshots.
- Add Playwright recording.

## Pull Request Guidelines

- Keep changes focused.
- Include a short explanation of the problem and solution.
- Run `npm run check`.
- Run `npm run build`.
- Do not include real secrets, tokens, cookies, or production traffic.

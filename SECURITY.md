# Security Policy

AITestAutomate can process browser traffic, headers, cookies, tokens, request bodies, and response bodies.

Use it carefully.

## Current MVP Safety Rules

- Do not paste production tokens.
- Do not run generated tests against production systems.
- Review high-risk endpoints before enabling tests.
- Avoid uploading traffic that contains sensitive personal data.

## Reporting Security Issues

This project is in early local MVP stage.

If you find a security issue, please open a private report when the GitHub repository is available, or contact the maintainers directly.
